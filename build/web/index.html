¿Cómo funciona?

1. Los usuarios registrados pueden cifrar mensajes y generar un token único

2. Comparte el token con quien quieras descifrar el mensaje

3. Cualquier persona (incluso sin registro) puede descifrar usando el token

4. Cada token es válido por 7 días y solo puede usarse una vez
